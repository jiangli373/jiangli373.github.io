ubuntu server下配置UTF8中文的步骤

ubuntu

第一步，安装中文包： 

sudo apt-get install language-pack-zh-hant-base language-pack-zh-hans-base

第二步，配置相关环境变量：

sudo vi /etc/environment

LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh:en_US:en"

第三步，重新配置本地配置

sudo dpkg-reconfigure locales
